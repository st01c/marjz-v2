For use with pure text mini CMS editor.html:

To run local server:

1) Terminal >
cd /Users/martinzeilinger/Documents/Websites/marjz-v2 && python3 -m http.server 8000

2) Browser >
http://localhost:8000/index.html (browse draft site)

3) To load pages directly:
Detail pages: http://localhost:8000/item.html?id=... (ids/slugs from content.json)

4) For editor:
Editor: http://localhost:8000/editor.html (to load/edit/download data/content.json)

5) Replace updated content.json file with downloaded file.

===

For use with Decap:
Three-way split terminal window

Terminal 1
node scripts/refresh-decap-options.js && npx decap-server
(updates type/tag options, then runs Decap UI server)

Terminal 2
python3 -m http.server 8080
(runs local site server)

Terminal 3:
node scripts/build-content.js
(after saving new entries (or changed entries), run to rebuild site)


Access:
http://localhost:8080/ [site]

http://localhost:8080/admin/ [admin UI]
